About CodeLighthouse

Abstract

CodeLighthouse is a web application that allows users to hone their coding skills in a pleasant, community-based way. The users are able to check their proposed solutions for coding challenges that are offered by the application, directly into their browser. The platform offers automatic solution verifying and checking. In addition, the application offers support for creating groups of people (referred to as Lighthouses in the following) that can either be Private (joining will be done with the help of a randomly generated code), or Public (referred to as Communities in the following). The enrollment code of a community is public and can be used to join the group. One proposed usage scenario for communities is to be used as a way of checking out what certain institutions are up to (e.g: Universities, Big tech companies (via interview questions)).

Running contests on the platform is also possible. An admin can create either a private contest and upload a custom participants list to then have accounts automatically created and sent(via e-mail) to each and every one of them or a public contest (open contest), that allows any user on the platform to participate. One identified usage scenario is for tech companies that want to run talent acquisition-like campaigns.

The solution is initially presented from an architectural perspective that is followed by the features and use cases of the application.

Objectives

CodeLighthouse wants to combine the best of 3 worlds:

  • Online Computer Science coding platforms (e.g: CodeWars, LeetCode, HackerRank).

  • Online courses that also allow users to run code in an interactive code editor (e.g: Stepik).

  • Online contests for companies (e.g: HackerRank).

Basic Concepts

Challenge difficulty

Because CodeLighthouse wants to enlighten people's skills in coding, the main grading and ranking system uses the following names:

  • Dark 5 -> 1

  • Light 1 -> 5


    There is no category in between light and dark.

Terminology

CodeLighthouse employs a few inspired names:

  • Lighthouse - A group of programmers (like a community)

  • Communities - A public, open, lighthouse

  • Challenge - A computer-science coding problem or question that awaits your skills
