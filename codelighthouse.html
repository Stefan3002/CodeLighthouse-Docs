About CodeLighthouse

CodeLighthouse wants to combine the best of 3 worlds:

  • Online Computer Science coding platforms (e.g: CodeWars, LeetCode, HackerRank).

  • Online courses that also allow users to run code in an interactive code editor (e.g: Stepik).

  • Online live one-to-one coding interviews for companies (e.g: HackerRank).

Basic Concepts

Because CodeLighthouse wants to enlighten people's skills in coding, the main grading and ranking system uses the following names:

  • Dark 5 -> 1

  • Light 1 -> 5


    There is no category in between light and dark.
