Creating Challenges

Introduction

Whenever you want to create a new challenge, you have to abide by some rules. Failing to do so, will crash the app on your side and you will not be able to create a new challenge

In order to make your life easier as a user, the app was designed to abstract the testing library used to assert the outputs. This is why you need to provide the app with 2 main functions:

  • true function

  • random function

Requirements

Here is a simple example of how a true function and a random function should look like:

def true_function(inputs): a = inputs[0] b = inputs[1] return a - b
def random_function(): tests = [] for i in range(100): tests.append((i, i + 10)) return tests
