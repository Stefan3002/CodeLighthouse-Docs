Verdicts on challenges

As an admin, it is your duty to giver verdicts on pending challenges. You will only be able to see challenges that were not made private. The only user that can see private challenges is the developer who has access to the database. Let's respect our users privacy shall we?

Types of verdicts

  • Approve

    • Approving a challenge will make it immediately public. Admins, please, make sure it complies with the code of conduct of CodeLighthouse.

  • Send back

    • Sending back a challenge will signal to the original author that there are some things that are not alright.

  • Deny

    • Denying a challenge is a powerful action: it will forever remove the challenge from the platform. No appeal right. Please, admins, use this one carefully. Only use it when the challenge is very broken to the point where it is better to start over.
